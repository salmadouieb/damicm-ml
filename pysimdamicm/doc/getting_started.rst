=========================
PySimDamicmM Introduction
=========================

Pysimdamicm is a Python3 module for **data acquisition process** of the DAMIC-M detector, 
**as well as for the reconstruction process of RAW data** (either SIM or real data). 

This is the documentation for PySimDAMICM version 1.0.1.

See **more documentation** about Geant4 simulations and reconstruction process at `notes
<https://www.overleaf.com/project/5d5d2e5ccc8d6e01a4381c64>`_
